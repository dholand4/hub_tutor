Gerador de Questões para Moodle (Formato XML)

Como Utilizar o Sistema

Este sistema permite gerar questões no formato XML compatível com o Moodle. Para utilizá-lo corretamente, siga as instruções abaixo:

  • Digite suas questões no campo de texto, utilizando uma linha para cada pergunta.
  • Cada questão deve começar com a numeração sequencial, como 1., 2., 3., e assim por diante.
  • Inclua as alternativas com as letras de a) até e), uma por linha, logo abaixo da respectiva questão.
  • Marque a alternativa correta adicionando a tag {correta} ou {correto} ao final da alternativa correta.
  • Exemplo de questões: 
    1. Qual é a capital do Brasil?
a) São Paulo
b) Rio de Janeiro
c) Brasília {correta}
d) Salvador
e) Belo Horizonte

2. Qual é a capital do estado do Ceará?
a) Sobral
b) Juazeiro do Norte
c) Crato
d) Fortaleza {correto}
e) Quixadá

Após inserir todas as questões, clique no botão "Gerar XML" para obter o arquivo no formato adequado para importação no Moodle.
